## 3. QUICK REFERENCE SENSORY GRADING CARD

**Purpose**: Field reference for sensory assessment
**Format**: Laminated card for use during inspection

```
SENSORY QUALITY GRADING - QUICK REFERENCE
═════════════════════════════════════════════════════════════

GOOD (G) - ACCEPT
─────────────────────────────────────────────────────────────
Appearance:  Translucent, shiny, firm, moist
             Proper color for species, bright
Odor:        Fresh ocean air, clean seaweed
             No decomposition or aged odors
Flavor:      Sweet, salty, neutral, characteristic
             No sour, ammonia, or off-notes
Texture:     Firm, moist, resilient
             No mushiness or breakdown


REASONABLY GOOD (RG) - ACCEPT
─────────────────────────────────────────────────────────────
Appearance:  Slightly opaque, slight dulling
             Color slightly faded but acceptable
Odor:        Neutral to slightly fishy
             Slight aged/freezer odor
Flavor:      Neutral to slightly aged
             Acceptable characteristics
Texture:     Slightly firm, slightly moist
             Minor texture variation


FAIR (F) - HOLD/REVIEW
─────────────────────────────────────────────────────────────
Appearance:  Moderately opaque, surface dry
             Faded color, minor texture issues
Odor:        Moderate fishy, cardboardy, oxidized
             Slight musty or sulfide notes
Flavor:      Moderate aged characteristics
             Slight off-notes developing
Texture:     Slightly soft, slightly dry
             Some mushiness detectable


POOR (P) - REJECT
─────────────────────────────────────────────────────────────
Appearance:  Strongly opaque, grainy, mushy
             Yellowing, browning, dull
Odor:        Sour, ammonia, fecal, putrid
             Strong rancid, moldy, fermented
Flavor:      Sour, rancid, sickly sweet
             Strong ammonia, sulfur notes
Texture:     Mushy, soft, grainy, pasty
             Loss of structure, desiccated
```

```
  ✓ Labels complete and readable
  ✓ No temperature abuse signs

  Temperature: 0.5°F          → PASS
  Glaze: 15%                  → PASS (10-20% acceptable)
  Count: 21/25 per pound      → PASS (matches specification)

  Color: Pink/translucent     → GOOD
  Odor: Fresh, clean          → GOOD
  Texture: Firm, moist        → GOOD
  No visible defects          → GOOD

Inspector Decision: ACCEPT ✓

```

### SCENARIO 2: Crab Legs - Hold for Review

```
CASE: King Crab Legs - Quality Concern

Initial Assessment:
  ✓ Trailer temperature: -1°F (PASS)
  ⚠ Cases slightly damp (refrozen? TBD)
  ✓ Labels intact

Detailed Inspection:
  Temperature: -0.5°F         → PASS
  Glaze: 8%                   → FLAG (below 10% minimum)
  Workmanship: Good           → PASS

Sensory Evaluation:
  Color: Good                 → GOOD
  Odor: Fresh                 → GOOD
  Texture: Firm, moist        → GOOD

Issue: Glaze below specification indicates possible dehydration risk
Defect Assessment: No visible dehydration, but glaze loss concerning

Inspector Decision: HOLD - Requires supervisor evaluation
Supervisor Review: CONDITIONAL ACCEPT
Condition: Reduce price per glaze shortfall (estimated 1-2% weight loss)

Result: Product accepted at reduced price, documented for supplier
```

```
  Odor: AMMONIA detected      → CRITICAL RED FLAG
  Color: Slight discoloration
  Texture: Slightly mushy
  Grade: POOR

This is an automatic REJECT regardless of temperature

Inspector Decision: REJECT ✗
Supervisor Review: CONFIRMED REJECT ✗

Result: Product returned to supplier, documented as spoiled
Notification: Supplier contacted regarding quality issue
```

## 8. FIELD INSPECTOR QUICK START GUIDE

```
SEAFOOD QC INSPECTION - QUICK START
═════════════════════════════════════════════════════════════

5-MINUTE RECEIVING CHECK
─────────────────────────────────────────────────────────────
1. Take temperature:       _____ °F
   ✓ PASS if within limits for product type
   ✗ FAIL = Hold shipment

2. Check label:           [ ] Complete  [ ] Missing info
   ✗ FAIL = Hold for label correction

3. Inspect packaging:     [ ] Intact  [ ] Damaged
   ✗ FAIL = Assess product exposure

4. Smell product:         [ ] Fresh   [ ] Off odor
   ✗ FAIL = Reject immediately

5. Look inside:           [ ] Good     [ ] Issues visible
   If PASS on 1-4 → Proceed to detailed inspection
   If ANY FAIL → Hold/Reject decision

15-MINUTE DETAILED INSPECTION
─────────────────────────────────────────────────────────────
□ Weight & glaze (if frozen)
□ Size/count verification
□ Visual quality assessment
□ Sensory evaluation (smell, look)
□ Defect check & documentation
□ Fill out full inspection form
□ Take photos if defects

DECISION POINT
─────────────────────────────────────────────────────────────
All Criteria Pass?
  ✓ YES → Grade: Good/Reasonably Good → ACCEPT
  ⚠ SOME → Grade: Fair → HOLD for supervisor
  ✗ NO → Grade: Poor → REJECT

Red Flags = Automatic Reject:
  □ Ammonia/sour/putrid smell
  □ Soft/mushy texture
  □ Temperature abuse
  □ Missing required labels
  □ Foreign material
  □ Mold/visible spoilage
```
