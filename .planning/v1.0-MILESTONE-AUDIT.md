### Integration Paths Verified

1. **Clerk Auth → All Features**
   - `App.tsx` → `ClerkProvider` → `ConvexProviderWithClerk` → `ClerkAuthProvider`
   - `ClerkAuthProvider` populates `AuthContext` with userId, orgId, role
   - All features consume via `useAuth()` hook

2. **Form Builder → Worker Forms**
   - Admin creates template → `useTemplatePersistence`
   - `createTemplate`/`publishTemplate` → Convex formTemplates
   - `useTemplateSync` → queries `listPublishedTemplates`
   - Results sync to Dexie → `FormList` reads from Dexie

3. **Form Filling → Review Workflow**
   - Worker fills form → `FormFillingPage` → IndexedDB submission
   - `triggerSync()` → sync worker processes queue
   - `createSubmission` mutation → Convex
   - `ReviewerDashboard` → queries `listPendingSubmissions`

4. **Review Actions → Worker Status**
   - Reviewer approves/rejects → Convex mutations
   - `WorkerStatusList` → queries `listWorkerSubmissions`
   - Real-time updates via Convex reactivity
