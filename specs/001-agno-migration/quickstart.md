# Quickstart: Agno Framework Migration

**Feature**: 001-agno-migration
**Purpose**: Get the Agno-powered agent running locally

## Prerequisites

- Python 3.11+
- UV package manager
- Redis running (via docker-compose)
- OpenRouter API key

## 1. Install Dependencies

```bash
cd backend

# Add Agno framework
uv add agno

# Remove Anthropic SDK (no longer needed for agent)
uv remove anthropic

# Verify installation
uv run python -c "from agno.agent import Agent; print('Agno installed')"
```

## 2. Configure Environment

Add to `backend/.env`:

```bash
# OpenRouter Configuration
OPENROUTER_API_KEY=sk-or-v1-your-key-here
OPENROUTER_MODEL_ID=anthropic/claude-3.5-sonnet

# Keep existing variables
SUPABASE_URL=...
SUPABASE_KEY=...
REDIS_URL=redis://localhost:6379
```

## 3. Verify Agent Works

Create a test script `backend/test_agent.py`:

```python
from agno.agent import Agent
from agno.models.openrouter import OpenRouter
from agno.media import Image

# Initialize agent
agent = Agent(
    model=OpenRouter(id="anthropic/claude-3.5-sonnet"),
    markdown=True,
)

# Test text-only
print("=== Text Test ===")
agent.print_response("What is 2 + 2?", stream=True)

# Test with image (use any public image URL)
print("\n=== Vision Test ===")
image = Image(url="https://upload.wikimedia.org/wikipedia/commons/thumb/0/0b/Cat_poster_1.jpg/1200px-Cat_poster_1.jpg")
agent.print_response("What do you see in this image?", images=[image], stream=True)
```

Run it:

```bash
cd backend
uv run python test_agent.py
```

## 4. Test Tool Definition

Create `backend/test_tools.py`:

```python
from agno.agent import Agent
from agno.models.openrouter import OpenRouter
from agno.tools import tool

@tool(
    name="show_confirmation_modal",
    description="Display extracted data for user confirmation",
    stop_after_tool_call=True
)
def show_confirmation_modal(data: dict) -> str:
    """Store confirmation data for client retrieval."""
    print(f"TOOL CALLED: show_confirmation_modal with {data}")
    return "confirmation_123"

agent = Agent(
    model=OpenRouter(id="anthropic/claude-3.5-sonnet"),
    tools=[show_confirmation_modal],
    markdown=True,
)

# This should trigger the tool
agent.print_response(
    "I have data to confirm: weight=1250kg, batch=B-2024-001. Please show the confirmation modal.",
    stream=True
)
```

Run it:

```bash
cd backend
uv run python test_tools.py
```

## 5. Run the Backend

```bash
cd backend
uv run uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## Common Issues

### "OPENROUTER_API_KEY not set"
Ensure `.env` file exists in `backend/` directory and contains the key.

### "Model not found"
Check the model ID is valid on OpenRouter. Try `anthropic/claude-3.5-sonnet`.

### "Tool not called"
Ensure tool is registered in the `tools=[]` parameter when creating the Agent.

### "Image processing failed"
Verify the model supports vision (Claude, GPT-4o do; some models don't).
