<task type="auto">
  <name>Task 1: Create Docker Compose file for Convex self-hosted deployment</name>
  <files>deploy/convex/docker-compose.yml</files>
  <action>
Create `deploy/convex/docker-compose.yml` with two services:

1. **backend** service:
   - Image: `ghcr.io/get-convex/convex-backend:latest`
   - Ports: 3210:3210 (API), 3211:3211 (HTTP actions)
   - Volume: `data:/convex/data` for persistent storage
   - Environment: `CONVEX_CLOUD_ORIGIN=${CONVEX_CLOUD_ORIGIN:-http://127.0.0.1:3210}`
   - Healthcheck: `curl -f http://localhost:3210/version` with interval 5s, start_period 10s
   - stop_grace_period: 10s, stop_signal: SIGINT

2. **dashboard** service:
   - Image: `ghcr.io/get-convex/convex-dashboard:latest`
   - Port: 6791:6791
   - Environment: `NEXT_PUBLIC_DEPLOYMENT_URL=${CONVEX_CLOUD_ORIGIN:-http://127.0.0.1:3210}`
   - depends_on backend with condition: service_healthy

3. Named volume `data` for persistence.

Follow the exact pattern from the official Convex repo: https://github.com/get-convex/convex-backend/blob/main/self-hosted/docker/docker-compose.yml
  </action>
  <verify>

  <name>Task 2: Deploy to Coolify and generate admin key</name>
  <action>Deploy Convex to Coolify infrastructure and generate admin key</action>
  <instructions>
1. **Create Coolify resource:**
   - Go to Coolify Dashboard (http://217.15.164.63:8000)
   - Create a new Docker Compose resource
   - Paste the contents of `deploy/convex/docker-compose.yml`
   - Set environment variable: `CONVEX_CLOUD_ORIGIN=https://<your-convex-domain>` (use Coolify-assigned domain or configure a custom one)

2. **Configure persistent storage:**
   - In Coolify resource settings, ensure the `data` volume is mapped to persistent storage

3. **Deploy:**
   - Click Deploy in Coolify
   - Wait for both backend and dashboard to show as healthy

4. **Generate admin key:**
   - Go to the resource's Terminal in Coolify
   - Run: `docker compose exec backend ./generate_admin_key.sh`
   - Copy the output key (format: `convex-self-hosted|01xxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`)
   - Save securely - needed for Plan 02

5. **Verify dashboard:**
   - Open the dashboard URL (port 6791) in browser
   - Confirm it loads and shows the Convex dashboard

When done, provide:
- The Convex backend URL (e.g., `https://convex.217.15.164.63.sslip.io`)
- The admin key
