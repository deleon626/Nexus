## 📋 Quick Reference: Critical Limits

### Temperature (Top Priority)
- Frozen Seafood: ≤ 0°F
- Refrigerated Cooked/RTE: ≤ 40°F (optimal 32-38°F)
- Molluscan Shellfish: ≤ 10°F

### Quality Assessment
- **Odor**: Fresh, no ammonia/sour smells
- **Color**: Species-appropriate, no discoloration
- **Texture**: Firm and moist (not mushy)
- **Packaging**: Intact, properly labeled
- **Dehydration**: Zero tolerance for significant loss

### Sensory Grading Scale
- **Good** → Accept (premium product)
- **Reasonably Good** → Accept (standard product)
- **Fair** → Hold/conditional (food service)
- **Poor** → Reject (not for human consumption)
